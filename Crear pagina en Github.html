Tutorial para crear una pagina web en el repositorio Github

Paso 1

Crearemos una cuenta en el sitio web oficial del repositorio Github

Paso 2

Una vez creada la cuenta, vamos a crear un nuevo repositorio

¡ATENCIÓN!

Es importante que a la hora de crear el repositorio este tenga el nombre de usuario de nuestra cuenta, además debera terminar en github.io

Por ejemplo: Cuenta.github.io

Paso 3

Si hicimos todo bien hasta ahora, deberiamos estar en nuestro repositorio, ahora subiermos los archivos de nuestra pagina web

Es importante aclarar que su repositorio no contara con archivos ya existentes, los que se muestran son del repositorio en el cual esta creada esta pagina web

En esta seccion subiermos todos los archivos y carpetas de nuestra pagina web

Es recomendable que el archivo principal de nuestra pagina web tenga el nombre index.html

Paso 4

Una vez subidos todos los archivos de nuestra pagina, Github tardara unos momentos en procesar y generar nuestra pagina, si queremos comprobar que nuestra pagina ya se encuentra activa ingresaremos la url de nuestro repositorio en nuestro navegador web, en el caso de este repositorio: Emilianuu.github.io

Y con eso ya tendriamos nuestra pagina creada con Github
